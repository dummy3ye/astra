# Bot Commands

## Moderation

| Command       | Slash | Description                     |
| ------------- | ----- | ------------------------------- |
| `/ban`        | ✅    | Ban a user with optional reason |
| `/kick`       | ✅    | Kick a user                     |
| `/warn`       | ✅    | Warn a user                     |
| `/warnings`   | ✅    | List warnings for a user        |
| `/clearwarns` | ✅    | Clear all warnings for a user   |
| `/timeout`    | ✅    | Timeout a user (duration)       |
| `/untimeout`  | ✅    | Remove timeout                  |
| `/purge`      | ✅    | Bulk delete messages            |

## Leveling

| Command        | Slash | Description                      |
| -------------- | ----- | -------------------------------- |
| `/rank`        | ✅    | Show your rank or another user's |
| `/leaderboard` | ✅    | Top XP rankings                  |

## Automod

| Command                 | Slash | Description                         |
| ----------------------- | ----- | ----------------------------------- |
| `/automod blocklinks`   | ✅    | Enable/disable link blocking        |
| `/automod blockedwords` | ✅    | Set blocked words (comma-separated) |

## Settings

| ----------- | ----- | ------------------------- |
| `/settings` | ✅    | View server configuration |

## Utility

| Command | Slash | Description       |
| ------- | ----- | ----------------- |
